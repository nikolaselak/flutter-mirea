# Первая практика ПКС

### Install

```bash
vim $HOME/.zshrc
export PATH=$HOME/YOUR_FLUTTER_DIR/flutter/bin
```

### Usage

Create

```bash
flutter create .
```

Run

```bash
flutter run
```
